# 2.2 迭代法求方程根

## 题目
用迭代法求方程 e^x + 10x - 2 = 0 的根，要求精度为 10⁻⁴。

## 方法
采用两种迭代方法：

### 1. 牛顿迭代法
迭代公式：x_{n+1} = x_n - f(x_n)/f'(x_n)

其中：
- f(x) = e^x + 10x - 2
- f'(x) = e^x + 10

### 2. 简单迭代法（不动点迭代）
将原方程变形为：x = (2 - e^x) / 10

迭代公式：x_{n+1} = (2 - e^{x_n}) / 10

## 结果

### 牛顿迭代法
- **迭代次数**: 2次
- **根的近似值**: **0.0905251013**
- **f(0.0905251013) = 1.0854428467e-11**
- **精度**: 4.45×10⁻⁶ < 10⁻⁴

### 简单迭代法
- **迭代次数**: 4次
- **根的近似值**: **0.0905264681**
- **f(0.0905264681) = 1.5163698044e-5**
- **精度**: 1.39×10⁻⁵ < 10⁻⁴
