# 2.1 二分法求方程根

## 题目
用二分法求方程 x⁴ - 3x + 1 = 0 在区间 [0.3, 0.4] 内的根，要求误差不超过 10⁻²。

## 方法
二分法（Bisection Method）

## 结果
- 根的近似值: **0.343750**
- f(0.343750) = -0.017287
- 最终误差: 0.05
